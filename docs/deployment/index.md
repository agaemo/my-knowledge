# Deployment

リリース戦略とデプロイの安全性を高める手法。

- [カナリアリリース](/deployment/canary-release) — 一部のユーザーに段階的に展開する戦略
- [ブルーグリーン](/deployment/blue-green) — 2環境を切り替えてダウンタイムなくリリース
- [フィーチャーフラグ](/deployment/feature-flags) — コードを変えずに機能の有効・無効を切り替える
- [エッジコンピューティング](/deployment/edge-computing) — ユーザーに近い場所で処理を実行するアーキテクチャ
- [CDN](/deployment/cdn) — 分散エッジサーバーでコンテンツをキャッシュ・配信しレイテンシを削減する仕組み
