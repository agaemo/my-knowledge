# 環境管理

開発環境の再現性・バージョン管理を扱うツール。

- [Nix](/dev-tools/env-management/nix) — 純粋関数型アプローチで環境を宣言的・再現可能に管理するパッケージマネージャー
- [mise](/dev-tools/env-management/mise) — プロジェクトごとにツールバージョン・環境変数・タスクを一元管理する開発環境マネージャー
- [Dev Container](/dev-tools/env-management/dev-container) — Docker コンテナを開発環境として使い、OS レベルの環境差異をなくす仕組み
