# Database

データベースの概念・データモデル・アーキテクチャパターン。

- [RDBMS](/platforms/database/rdbms) — リレーショナルDB。ACID保証・行志向・正規化（PostgreSQL・MySQL）
- [NoSQL](/platforms/database/nosql) — ドキュメント・KVS・カラム型・グラフのデータモデルと使い分け
- [NewSQL](/platforms/database/newsql) — 水平スケールとACIDトランザクションを両立する分散SQL（CockroachDB・TiDB・PlanetScale）
- [インメモリDB](/platforms/database/in-memory-db) — RAMにデータを保持する高速データストア。キャッシュ・セッション・Pub/Sub（Redis）
- [OLAP（分析DB）](/platforms/database/olap) — 列志向ストレージで集計クエリを高速化。組み込み型のDuckDB・クラウド型のBigQuery
- [ACID とトランザクション](/platforms/database/acid) — 原子性・一貫性・分離性・永続性の4性質と分離レベルの使い分け
