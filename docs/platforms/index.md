# Platforms

クラウド・データベース・BaaSなど、乗っかって使うマネージドサービス・プラットフォーム。

- [クラウド](/platforms/cloud) — オンプレミスとの違い・IaaS/PaaS/SaaSのモデル・主要プラットフォームの特徴
- [XaaS](/platforms/xaas) — IaaS・PaaS・SaaS・FaaS・CaaS・BaaS・DaaS の責任分担モデル
- [Database](/platforms/database/) — NoSQL・NewSQLのデータモデルとアーキテクチャ
- [BaaS](/platforms/baas) — 認証・DB・ストレージをまとめて提供するバックエンドサービス（Firebase・Supabase）
- [Apache Kafka](/platforms/kafka) — イベントストリーミング基盤。トピック・パーティション・オフセット管理の設計
