# AI・機械学習・LLM の全体像

## AI の系譜

```
AI（人工知能）
└── 機械学習（Machine Learning）
    └── 深層学習（Deep Learning）
        └── LLM（大規模言語モデル）
```

AI は広い概念で、機械学習はその手法のひとつ。深層学習は機械学習の中でも特にニューラルネットワークを深く積み重ねる手法。LLM はその深層学習を大規模なテキストデータで訓練した言語特化モデル。

---

## AI（人工知能）

コンピュータに人間のような知的な振る舞いをさせること全般を指す。定義が広く、チェスの探索アルゴリズムも AI と呼ばれる時代があった。

現在は「機械学習でデータから学習するシステム」を指すことが多い。

---

## 機械学習（Machine Learning）

**ルールを人間が書かずに、データからパターンを学習させる手法。**

従来のプログラミングとの対比：

```
従来: ルール（if/else）を人間が書く → 入力 → 出力
機械学習: 大量の入力・出力のペアを与える → モデルがルールを学習
```

### 学習の種類

**教師あり学習（Supervised Learning）**
正解ラベル付きのデータで学習する。スパムメール判定・画像分類・価格予測など。

**教師なし学習（Unsupervised Learning）**
正解ラベルなしで、データの構造・パターンを発見する。クラスタリング・次元削減・異常検知など。

**強化学習（Reinforcement Learning）**
エージェントが環境と相互作用しながら、報酬を最大化する行動を学習する。ゲームの攻略・ロボット制御・推薦システムの最適化など。

```
エージェント → 行動 → 環境 → 報酬 → エージェント（学習）
```

---

## 深層学習（Deep Learning）

**多層のニューラルネットワークを使った機械学習。**

人間の脳の神経回路を模した「ニューロン」を何層にも積み重ね、複雑なパターンを自動的に抽出できる。

2012年の ImageNet コンペで深層学習が圧倒的な精度を出したことで注目が爆発し、それ以降の AI ブームを牽引した。

### 従来の機械学習との違い

| | 従来の機械学習 | 深層学習 |
|---|---|---|
| 特徴量 | 人間が設計する（特徴エンジニアリング） | モデルが自動で抽出する |
| データ量 | 少量でも動く | 大量のデータが必要 |
| 計算資源 | 軽い | GPU が必須 |
| 解釈性 | 比較的高い | ブラックボックスになりやすい |

### 主なアーキテクチャ

- **CNN（畳み込みニューラルネットワーク）** — 画像認識に強い。空間的な特徴を捉える
- **RNN / LSTM** — 時系列・テキストなど順序のあるデータを扱う。LLM 以前の主流
- **Transformer** — 2017年登場。自己注意機構（Attention）で長距離の依存関係を捉える。現代の LLM の基盤

---

## LLM（大規模言語モデル）

**Transformer アーキテクチャを、インターネット規模のテキストで大規模に訓練した言語モデル。**

GPT・Claude・Gemini などが代表。「大規模」とは、パラメータ数（数十億〜数兆）とデータ量（数兆トークン）の両方を指す。

### それ以前の AI との決定的な違い

| | 従来の AI / ML | LLM |
|---|---|---|
| 汎用性 | タスク特化（分類・検出・翻訳など1つ） | 1モデルで多様なタスク |
| 指示方法 | モデルの再学習が必要 | 自然言語で指示できる（プロンプト） |
| 文脈理解 | 限定的 | 長い文脈を保持して推論できる |
| 創造性 | ほぼなし | 文章・コード・アイデアを生成できる |

### なぜこれほど汎用的になったか

「次のトークンを予測する」という単純なタスクを超大規模に学習することで、言語の構造だけでなく世界の知識・論理・コードなどが内部表現として獲得された。これが「創発（Emergence）」と呼ばれる現象で、規模が増すにつれて予期せぬ能力が生まれた。

### RLHF（人間のフィードバックによる強化学習）

事前学習済みの LLM を「人間に役立つ回答をする」方向に調整する手法。
強化学習を活用し、人間の評価を報酬として fine-tuning する。ChatGPT の実用性を高めた重要な技術。

```
事前学習済み LLM → 人間がランク付け → 報酬モデル学習 → 強化学習で調整 → ChatGPT 相当のモデル
```

## まとめ：どこで何を使うか

| 技術 | 向く問題 | 例 |
|---|---|---|
| ルールベース AI | 条件が明確・変化が少ない | スパムフィルタ（単純なもの）・チェスの序盤定石 |
| 機械学習 | 数値・表データの予測・分類 | 価格予測・不正検知・推薦スコア |
| 深層学習 | 画像・音声・高次元データ | 画像分類・物体検出・音声認識 |
| LLM | テキスト理解・生成・推論 | 文章生成・コード生成・Q&A・要約 |
| 強化学習 | 連続的な意思決定・最適化 | ゲーム攻略・ロボット制御・広告入札 |
