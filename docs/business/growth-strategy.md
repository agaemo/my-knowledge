# 成長戦略（SLG / PLG / MLG / CLG）

何をビジネス成長のエンジンにするかを表す概念。SaaS・スタートアップでの文脈で使われることが多い。

## SLG（Sales Led Growth）

**営業チームが成長の主エンジン。**

SDR が見込み客を探し、AE がクロージングし、CSM が継続・拡大を担う分業モデル（[The Model](/org/the-model)）がその典型。

```
認知 → リード獲得 → 営業接触 → デモ → 提案 → 契約 → 活用支援
              ↑                              ↑
         マーケティング                  営業が主役
```

**向いている場面:**
- 複雑な製品・高単価（エンタープライズ）
- 意思決定者が複数いる組織向け販売
- 顧客がどんな製品かを理解するのに説明が必要

**課題:** 営業コストが高い。スケールするには採用・育成が必要。

---

## PLG（Product Led Growth）

**プロダクト自体が獲得・活性化・拡大のエンジン。**

ユーザーがまず使ってみて、価値を体験してから課金する。フリートライアル・Freemium モデルが典型。

```
認知 → セルフサインアップ → プロダクトを使う → 価値体験 → 自然に課金
                                  ↑
                          プロダクトが主役
```

**代表例:** Slack・Figma・Notion・Dropbox・Zoom・Canva

**向いている場面:**
- セットアップが簡単で個人が試しやすい製品
- ユーザーがチームを招待することで広がる（バイラルループ）
- 個人→チーム→組織全体への拡大が自然に起きる

**課題:** フリーユーザーのインフラコスト。エンタープライズ拡販が難しいケースがある。

### PLG の指標

| 指標 | 内容 |
|---|---|
| **Activation Rate** | サインアップ後、価値を体験した割合 |
| **PQL**（Product Qualified Lead） | プロダクト上の行動から「購入可能性が高い」と判断されたユーザー |
| **Time to Value** | サインアップからAha Moment（価値体験）までの時間 |

---

## MLG（Marketing Led Growth）

**コンテンツ・ブランド・広告が成長エンジン。**

SEO・コンテンツマーケティング・SNS・広告で認知を作り、インバウンドリードを獲得する。

```
SEO記事 / SNS / 広告 → 認知 → 問い合わせ → 営業へ
```

**代表例:** HubSpot（コンテンツマーケティングの先駆者）

PLG・SLG と組み合わせて使うことが多い。単独のモデルというより、マーケティング投資で成長を加速させる戦術的な側面が強い。

---

## CLG（Community Led Growth）

**コミュニティが成長エンジン。**

製品を中心としたユーザーコミュニティが、口コミ・相互学習・貢献を通じて新規ユーザーを引き込む。

```
既存ユーザーがコミュニティに参加
  → 知識を共有・助け合い
  → コミュニティ外から新規ユーザーが引き寄せられる
  → コミュニティ自体が製品の価値を高める
```

**代表例:** Figma（デザイナーコミュニティ）・Notion（テンプレート共有）・HashiCorp（オープンソース）

**向いている場面:** ユーザー同士のノウハウ共有に価値がある製品。開発者向けツール・クリエイターツール。

---

## 組み合わせ戦略

現代のスタートアップは単一のモデルではなく、組み合わせて使うことが多い。

```
よくある組み合わせ:

PLG + SLG:
  PLG で個人・SMB を獲得 → 社内で広がる → エンタープライズはセールスが支援
  （Slack・Figma・Zoom がこのモデル）

MLG + PLG:
  コンテンツで認知 → セルフサインアップ → プロダクトで価値体験
  （Notion・Canva に近い）
```

どのモデルを選ぶかは「誰が意思決定者か」「製品の複雑さ」「単価」によって決まる。
