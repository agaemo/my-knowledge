# 資金調達

スタートアップが成長資金を外部から調達する仕組み。調達するたびに投資家に株式を渡すことで、会社の「所有権の分散（希薄化）」が起きる。

## 資金調達ステージ

事業の成熟度に応じてステージが進む。

```
アイデア段階
    ↓
エンジェルラウンド / Pre-Seed
    ↓
シードラウンド（Seed）
    ↓
Series A
    ↓
Series B
    ↓
Series C 以降
    ↓
IPO（上場）または M&A（Exit）
```

### エンジェルラウンド / Pre-Seed

プロダクトがまだない、またはごく初期の段階。個人投資家（エンジェル）や創業者の自己資金が中心。数百万〜数千万円規模。

### シードラウンド（Seed）

プロトタイプや初期ユーザーが存在する段階。PMF（製品市場適合）を模索するフェーズ。シード特化の VC やエンジェルが投資。

### Series A

PMF が見えてきた段階。再現性のある顧客獲得モデルを作り、スケールの準備をする。数億〜十数億円規模が多い。ここから本格的な VC が主な投資家になる。

### Series B 以降

スケールの実証ができた段階。組織拡大・新市場開拓・海外展開などに使う。調達額が大きくなり、より大きな VC やプライベートエクイティが参加することも。

### Exit（イグジット）

投資家・創業者・従業員が保有する株式を換金できる状態にすること。VC はファンドの期限（約10年）内に Exit して LP に返済する必要があるため、Exit は出資時から見据えられている。

Exit の主な形態は IPO と M&A の2つ。

### IPO（Initial Public Offering）

株式を証券取引所に上場し、一般投資家が売買できるようにする。

- 投資家・従業員がストックオプションを売却して現金化できる
- 上場後も事業を継続・拡大する（Exit しつつ会社は残る）
- 審査・開示義務・IR など上場維持のコストが生じる
- ロックアップ期間（上場直後に主要株主が売却できない期間、通常 180日）がある

日本では東証グロース・プライム、米国では NASDAQ・NYSE が主要な上場先。

### M&A（Merger & Acquisition）

他社に会社ごと売却（買収）されること。IPO と異なり、会社が独立した主体として残るとは限らない。

**買収の主なパターン：**

| パターン | 内容 |
|---|---|
| **戦略的買収** | 技術・顧客・市場を取得する目的。買収後も事業を継続させることが多い |
| **アクハイア**（Acqui-hire） | 技術より「チーム・人材」の獲得が主目的。プロダクトは廃止されることもある |
| **PE（プライベートエクイティ）買収** | 事業の効率化・再売却を目的とした金融的な買収 |

**創業者・従業員にとっての違い：**

```
IPO: 株を売るかどうかは自分で判断できる（ロックアップ後）
M&A: 交渉次第。アーンアウト条項（一定期間在籍・目標達成で追加報酬）がつく場合も
```

M&A は IPO より早期に実現できるケースが多く、スタートアップの Exit の件数としては M&A の方が圧倒的に多い。

## 投資家の種類

### エンジェル投資家（Angel Investor）

個人の資産を自ら投資する人。元起業家・経営者が多い。金額は小さいが、意思決定が速く、メンタリングも提供する。

### VC（Venture Capital）

ファンドを組成し、複数のスタートアップに投資する機関。LP（年金基金・機関投資家など）から資金を集め、約 10 年で運用・回収する。Exit（IPO・M&A）による大きなリターンを狙う。

### CVC（Corporate Venture Capital）

事業会社（トヨタ・ソニーなど）が設立したファンド。財務リターンに加え、自社のビジネスとのシナジーを重視する。

### アクセラレーター / インキュベーター

初期スタートアップを短期間（3〜6ヶ月）で集中支援するプログラム。Y Combinator・500 Startups が有名。少額出資＋メンタリング・ネットワーク提供が多い。

## 主要な概念

### バリュエーション（Valuation）

会社の評価額。投資家が「この会社はいくらの価値か」と判断した額。売上や利益ではなく、将来の成長期待で決まることが多い。

```
Post-money Valuation = Pre-money Valuation + 調達額
例: 評価額 10億円 の会社が 2億円調達 → Post-money = 12億円
    投資家の持ち分 = 2 / 12 ≈ 16.7%
```

### 希薄化（Dilution）

資金調達のたびに新株を発行することで、既存株主の持ち株比率が下がること。創業者も調達を重ねるほど持ち分が小さくなる。

### キャップテーブル（Cap Table）

誰が何%の株式を持つかを示した表。創業者・投資家・ストックオプションプールなどが記載される。

### ストックオプション（Stock Option）

従業員が将来一定価格で自社株を購入できる権利。スタートアップが現金給与の代わりに提供することが多い。Exit 時に価値が顕在化する。
