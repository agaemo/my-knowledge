# 会社の種類

## スタートアップ vs ベンチャー

日本では混同されることが多く、厳密な定義は業界でも揺れている。ただし日本の文脈でよく語られる区別として以下がある。

**スタートアップ**（Startup）  
新しい技術・市場・ビジネスモデルを生み出すことを前提とした企業。**既存概念の改善ではなく、新規概念の創出**が出発点になることが多い。  
急成長・スケールを目指し、VC やエンジェル投資家からの資金調達を通じて成長を加速させる。IPO または M&A による Exit を前提とした設計。

例：新しい AI プラットフォーム、前例のない SaaS モデル、新市場を切り開くサービス

**ベンチャー**（Venture）  
日本で 1970〜80年代から使われてきた言葉。**既存のビジネスモデル・市場に新規参入する企業**も含む広い意味で使われることが多い。スタートアップより成長速度や投資前提の縛りが緩い。

例：新しいレストランチェーン、地域に特化したコンサル会社、既存業界への新規参入企業

**注意点：** 両者に明確な定義はなく、日本の政策文書（経済産業省など）では「スタートアップ」を「急成長・イノベーション型の新興企業」の意味で使い、ベンチャーをその類義語として扱うことも多い。文脈によって使い分けられていると理解するのが現実的。

海外では "startup" が一般的で、"venture" は VC（ベンチャーキャピタル）の文脈で主に使われる。

## 時価総額による分類

スタートアップの規模・成長段階を時価総額（バリュエーション）で表す言葉。

| 名称 | バリュエーション | 由来 |
|---|---|---|
| **ユニコーン**（Unicorn） | 10億ドル（約1,500億円）以上 | 「伝説上の生き物のように希少」から |
| **デカコーン**（Decacorn） | 100億ドル以上 | Deca（10倍）+ Unicorn |
| **ヘクトコーン**（Hectocorn） | 1,000億ドル以上 | Hecto（100倍）+ Unicorn |

**ユニコーン企業の例**（過去・現在含む）: Airbnb・Stripe・SpaceX・Canva・SmartHR・Sansanなど

バリュエーションは実際の売上・利益ではなく、投資家が「将来この会社はこれだけの価値になる」と判断した時点の評価額。上場企業の時価総額とは異なる。

## ブートストラップ（Bootstrap）

外部からの資金調達をせず、自己資金や事業収益だけで成長するスタイル。

```
VC調達あり: 外部資金で急成長 → 株式を投資家に渡す → EXIT（IPO・M&A）を目指す
ブートストラップ: 自前で成長 → 株式を薄めない → 持続的な事業として続ける
```

SaaS 企業では "Bootstrapped SaaS" として意図的に選ぶケースもある。成長速度よりも収益性・自律性を優先する判断。

## ガゼル企業

年間成長率 20% 以上を継続する急成長企業。ユニコーンのようなバリュエーション基準ではなく、成長率で定義される。
