# 会社の種類

## スタートアップ vs ベンチャー

日本では混同されることが多いが、文脈によって使い分けられる。

**スタートアップ（Startup）**  
急成長を前提とした、スケーラブルなビジネスモデルを追求する初期段階の企業。  
テクノロジーを武器に短期間で市場を拡大することを目指す。資金調達（VC・エンジェル）を通じて成長を加速させるモデルが多い。

**ベンチャー（Venture）**  
日本固有のニュアンスが強い言葉。「新興企業」全般を指すことが多く、スタートアップより広い意味で使われる。成長速度や投資前提が必須ではない。

海外では "startup" が一般的で、"venture" は VC（ベンチャーキャピタル）の文脈で使われることが多い。

## 時価総額による分類

スタートアップの規模・成長段階を時価総額（バリュエーション）で表す言葉。

| 名称 | バリュエーション | 由来 |
|---|---|---|
| **ユニコーン（Unicorn）** | 10億ドル（約1,500億円）以上 | 「伝説上の生き物のように希少」から |
| **デカコーン（Decacorn）** | 100億ドル以上 | Deca（10倍）+ Unicorn |
| **ヘクトコーン（Hectocorn）** | 1,000億ドル以上 | Hecto（100倍）+ Unicorn |

**ユニコーン企業の例**（過去・現在含む）: Airbnb・Stripe・SpaceX・Canva・SmartHR・Sansanなど

バリュエーションは実際の売上・利益ではなく、投資家が「将来この会社はこれだけの価値になる」と判断した時点の評価額。上場企業の時価総額とは異なる。

## ブートストラップ（Bootstrap）

外部からの資金調達をせず、自己資金や事業収益だけで成長するスタイル。

```
VC調達あり: 外部資金で急成長 → 株式を投資家に渡す → EXIT（IPO・M&A）を目指す
ブートストラップ: 自前で成長 → 株式を薄めない → 持続的な事業として続ける
```

SaaS 企業では "Bootstrapped SaaS" として意図的に選ぶケースもある。成長速度よりも収益性・自律性を優先する判断。

## ガゼル企業

年間成長率 20% 以上を継続する急成長企業。ユニコーンのようなバリュエーション基準ではなく、成長率で定義される。
