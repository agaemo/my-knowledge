# KGI・KPI・OKR

目標設定と進捗計測のフレームワーク。「どこを目指すか（KGI）」「何を計測するか（KPI）」「どう達成するか（OKR）」と役割が異なる。

## KGI（Key Goal Indicator）

**最終的に達成したいゴールの指標。**

「売上 10億円」「シェア 30%」のように、事業・組織が最終的に目指す数値。  
KPI や OKR はすべて KGI に貢献するために設定される。

```
KGI の例:
  - 今期の ARR を 5億円にする
  - 顧客満足度（NPS）を業界トップ水準にする
  - 年間新規顧客 500社を獲得する
```

## KPI（Key Performance Indicator）

**KGI を達成するために動かすべきプロセスの指標。**

KGI が「結果」なら、KPI は「結果につながる行動・状態」を測る。  
KGI と KPI の因果関係を正しく設定することが重要。

```
KGI: ARR 5億円
  KPI（の例）:
    - 月間新規商談数
    - 商談から成約までの転換率
    - 平均契約単価
    - 解約率（チャーン）

  これらすべてが動くと ARR が上がる、という因果仮説を持つ
```

### KPI の条件

- **測定可能** — 数値として定期的に把握できる
- **行動と連動** — チームが意図的に動かせる
- **KGI への影響が明確** — 動かすと KGI に近づく根拠がある

### 良くある失敗

**虚栄の指標（Vanity Metrics）:** 「登録ユーザー数」「PV数」のように増えると嬉しいが、ビジネスの健全性と直結しない指標を KPI にしてしまう。実際に収益・継続・価値に結びつく指標を選ぶ。

## OKR（Objectives and Key Results）

**野心的な目標（O）と、その達成を示す定量的な成果（KR）を紐づけるフレームワーク。**  
詳細は [OKR](/business/okr) ページを参照。

KPI との違いは「モニタリング」か「変革」かにある。

| | KPI | OKR |
|---|---|---|
| 目的 | 日常業務の健全性を監視する | 次のステージへの変革を牽引する |
| 達成基準 | 100% 達成が基本 | 60〜70% 達成が理想（野心的な設定） |
| 期間 | 常時モニタリング | 四半期・年次 |
| 起点 | トップダウンが多い | ボトムアップ + トップダウンの組み合わせ |

## 3つの関係

```
KGI（最終ゴール）
  ↑ 達成を目指す
KPI（日常の健全性モニタリング）
  ↑ 行動を評価する
OKR（変革のための目標設定）
  ↑ 大きな変化を起こす
```

実務では「KGI を KPI で日々追いながら、四半期の OKR で飛躍を狙う」という組み合わせが多い。

## North Star Metric との関係

[North Star Metric](/business/north-star-metric) はプロダクトが生み出すコアバリューを表す単一指標。KGI・KPI の上位概念として「どの方向に進むべきか」を示す羅針盤の役割を持つ。

```
North Star Metric（方向性）
  ↓ を達成するために
KGI（期の最終ゴール）
  ↓ を監視するために
KPI（日常指標）+ OKR（変革目標）
```
