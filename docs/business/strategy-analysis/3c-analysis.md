# 3C分析

「Customer（市場・顧客）」「Competitor（競合）」「Company（自社）」の3つの視点で事業環境を分析するフレームワーク。日本の経営コンサルタント大前研一が提唱した。

## 3つの視点

```
        Customer（市場・顧客）
              ／  ＼
            ／      ＼
  Competitor ────────── Company
   （競合）              （自社）
```

- **Customer（市場・顧客）**: 市場規模・成長性、顧客のニーズ・購買行動の変化
- **Competitor（競合）**: 競合の数・シェア・強み弱み、新規参入の脅威
- **Company（自社）**: 自社の経営資源・強み弱み、ブランド・技術力

3つの円が重なる部分に、自社が獲りに行ける**KSF（Key Success Factor、重要成功要因）**が見えてくる、という考え方。

## なぜ存在するか

自社の強み・弱みだけを見て戦略を立てると、「強いと思っていたが市場のニーズとズレている」「競合も同じ強みを持っていて差別化にならない」といった誤りが起きる。

3C分析は、自社単独ではなく**市場・競合との関係性**の中で自社の立ち位置を捉えることを強制する。3つの視点をすべて埋めないと分析が完成しないため、視点の漏れを防ぐ。

## 分析の進め方

```
1. Customer: 市場規模・成長率・顧客セグメント・ニーズの変化を調べる
2. Competitor: 主要競合をリストアップし、シェア・強み弱み・戦略を整理する
3. Company: 自社の経営資源（ヒト・モノ・カネ・技術・ブランド）を整理する
4. 3つを重ね合わせ、「市場が求め、競合が提供できず、自社が提供できる」領域（KSF）を探す
```

実務では Customer → Competitor → Company の順に分析することが多い。先に市場を理解しないと、競合分析・自社分析の評価軸が定まらないため。

## SWOTとの違い・組み合わせ

| | 3C分析 | SWOT分析 |
|---|---|---|
| 軸 | 市場・競合・自社という3者の関係性 | 内部/外部 × ポジティブ/ネガティブの4象限 |
| 強み | 「自社が何で勝てるか」を関係性から導く | 強み・弱み・機会・脅威を網羅的に洗い出す |
| 弱み | 打ち手の具体化までは弱い | 3者の関係性（特に競合との差）は見えにくい |

3C分析で市場・競合・自社を分析し、その結果を[SWOT分析](/business/strategy-analysis/swot)の4象限（特に機会・脅威）に落とし込む、という順序で使われることが多い。

## いつ使うか

- 新規事業・新市場への参入を検討するとき
- 自社の強みが本当に「市場で評価され」「競合に対して優位」かを検証したいとき
- マーケティング戦略・事業戦略の前提となる環境分析をしたいとき

自社の内部要因に閉じた分析（技術力の棚卸しなど）には向かない。市場・競合という外部要素を含めた比較分析に向く手法。
