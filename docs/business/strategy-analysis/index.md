# 戦略分析フレームワーク

事業・市場・競合を構造的に把握し、戦略の方向性を導くための分析フレームワーク。意思決定の前提となる「現状把握」を担う。

- [3C分析](/business/strategy-analysis/3c-analysis) — 市場・競合・自社の3視点から戦略の打ち手（KSF）を導く
- [SWOT分析](/business/strategy-analysis/swot) — 強み・弱み・機会・脅威を整理し、内部要因と外部要因を掛け合わせる
- [ワードリーマップ](/business/strategy-analysis/wardley-map) — 価値連鎖と進化の度合いを地図化し、投資・外部化の判断を可視化する
