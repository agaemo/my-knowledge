# Platforms

クラウド・データベース・BaaS・CI/CDなど、マネージドサービス・プラットフォームのまとめ。

- [Cloud](/tools/cloud/) — オンプレミスとクラウドの違い・IaaS/PaaS/SaaSのモデル・主要プラットフォームの特徴
- [Database](/tools/database/) — サーバーレスDB・NewSQLなど、モダンなマネージドデータベースサービス
- [BaaS](/tools/baas/) — 認証・データベース・ストレージなどのバックエンド機能をマネージドで提供するプラットフォーム
- [CI/CD](/tools/ci-cd/) — コードの変更をトリガーにビルド・テスト・デプロイを自動化するプラットフォーム
