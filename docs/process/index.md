- [PoC・Prototype・MVP・Spike](/process/poc) — 本格開発の前に小さく試す手法の目的と使い分け
- [Git ブランチ戦略](/process/git-branching) — GitFlow・GitHub Flow・トランクベース開発の違いと選び方
- [コードレビュー](/process/code-review) — なぜ行うか・何を見るか・良いコメントの書き方・レビュー文化の設計
